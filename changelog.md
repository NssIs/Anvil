# Changelog

## Anvil v0.1.3

### Added

- Resource Pack AI assistant (Ollama, OpenRouter, or Google AI Studio): chat about your pack, attach images, and let it paint pixel edits as an undoable layer in the Pixel Editor.
- Shader Pack workspace: visually design a full Iris shader pack for Minecraft 1.21.x — lighting, shadows, water, fog, clouds, bloom, depth of field, and hundreds more options — and export it ready to play.
- Shader AI assistant: ask about your setup or let it change shader options for you.
- Shader code editor: GLSL syntax highlighting, file management, and automatic saving; files you hand-edit are never overwritten.
- Pixel Editor tools: fill, line, rectangle, ellipse, undo/redo, zoom and pan, eyedropper, and shortcuts for everything.

### Changed

- "Texture Pack" is now "Resource Pack".
- Smoother scrolling, drawing, and animations.

### Fixed

- None

### Notes

- Shader packs target Iris on Minecraft 1.21.x.
- Installers aren't OS-signed yet; the in-app updater verifies its own signatures.
