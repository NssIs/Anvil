About Anvil

Privacy policy

Your data stays on your computer

Anvil stores your projects, textures, and shader files locally on this computer. Anvil has no account system, no analytics, and no telemetry — we never collect, sell, or upload your data. There are no hidden background uploads: the only network requests Anvil makes are the ones listed below.

Where your data lives

Everything is kept in Anvil's application data folder: your project list, saved textures, exported packs' source data, shader files, and the AI provider settings. On Linux that is ~/.local/share/com.anvil.app; on Windows it is %APPDATA% and on macOS ~/Library/Application Support, under the same identifier. Deleting a project from Anvil's dashboard removes its files from that folder.

AI features

The AI assistant is optional and off until you configure a provider yourself (Ollama, OpenRouter, or Google AI Studio). When you send a message, your prompt, the relevant project data (such as shader options or texture pixels), recent conversation turns, and any images you attach are sent to that provider using your own API key. Nothing is sent anywhere when you don't use the assistant.

Your API key and model choice are stored locally in the app's storage — note that the key is stored unencrypted, so treat Anvil's data folder like any other folder containing secrets and prefer scoped, revocable keys. The key is never sent anywhere except to the provider it belongs to, as part of authenticating your own requests. The provider's own privacy policy and terms apply to everything you send through it. If you use a local model via Ollama, your AI requests never leave your machine at all.

Network access

At startup Anvil contacts GitHub to check for app updates (release metadata only) and to show the contributors list on the home screen. When you first open a resource pack project, Anvil downloads vanilla Minecraft textures from Mojang's public asset servers so you have something to edit. No personal data, identifiers, or usage information is sent with any of these requests.

Your choices

You can use Anvil fully offline — projects, the texture editor, and the shader builder work without a connection (the contributors list and update check will simply fail quietly). You can clear the AI configuration at any time from the AI setup dialog, and remove all of Anvil's data by deleting the application data folder listed above.

Updates

Anvil updates itself from signed GitHub releases. An update check downloads only release metadata; an actual update downloads the new app package and verifies its signature before installing.

Changes to this policy

If a future version of Anvil changes what data is stored or sent, this section will be updated in the same release. Because Anvil is open source, every change is publicly visible in the repository history.

Licensing

Anvil is MIT-licensed and open source. It is not affiliated with Mojang or Microsoft — Minecraft and its textures are © Mojang Studios. Questions or concerns are welcome on the GitHub issue tracker.

Full policies

The complete, versioned policy documents live in the repository: the privacy policy, the security policy (including how to report vulnerabilities privately), and the asset, content & trademark policy covering Minecraft assets and packs you share.
